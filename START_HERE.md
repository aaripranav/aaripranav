## 🐛 If Something Goes Wrong

### Page shows 404
```
Solution:
1. Go to Settings → Pages
2. Verify source is: main branch, / root folder
3. Check that index.html exists
4. Click Save
5. Wait 5 minutes
6. Hard refresh: Ctrl+Shift+R
```

### Changes don't appear
```
Solution:
1. Hard refresh: Ctrl+Shift+R
2. Clear browser cache
3. Try different browser
4. Wait 3-5 minutes for deployment
```

### Animations not working
```
Solution:
1. Hard refresh browser
2. Try different browser
3. Check browser console (F12)
4. Verify HTML file is correct
```

### Site not loading
```
Solution:
1. Check GitHub Actions tab
2. Look for failed builds
3. Verify Settings → Pages is configured
4. Wait longer (sometimes takes 5+ minutes)
```
